# Turbofan Engine RAG Chatbot

AI-powered chatbot for turbofan engine degradation analysis using the CMAPSS dataset.

## Technologies Used

- Streamlit
- ChromaDB
- Sentence Transformers
- Groq API (Llama 3)
- Pandas

## Features

- Retrieval-Augmented Generation (RAG)
- Semantic search using ChromaDB
- Llama 3 integration with Groq API
- Turbofan engine fault analysis
- Interactive chatbot UI
